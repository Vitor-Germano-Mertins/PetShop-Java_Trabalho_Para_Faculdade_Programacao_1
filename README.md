Sistema de gerenciamento de PetShop desenvolvido em Java utilizando os conceitos de Programação Orientada a Objetos (POO). O projeto realiza cadastro, busca, listagem e remoção de informações relacionadas a clientes, pets, funcionários e produtos do estoque.

O sistema possui diferentes tipos de pets (mamíferos, aves e peixes) e funcionários (gestores e veterinários), utilizando herança, polimorfismo e encapsulamento para organização das classes e reutilização de código.

Funcionalidades principais:
- Cadastro de clientes e seus pets
- Cadastro de funcionários
- Cadastro e gerenciamento de produtos no estoque
- Listagem de clientes, pets, funcionários e produtos
- Remoção de produtos do sistema
- Estrutura organizada em múltiplas classes e relacionamento entre objetos

Tecnologias utilizadas:
- Java
- ArrayList
- Scanner
- Programação Orientada a Objetos (POO)
